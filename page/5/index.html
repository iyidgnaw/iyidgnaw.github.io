慌乱的今天

发表于 2015-01-16   |   分类于 生活

今天上午考通原，本以为妥妥的能在考前做好十足的准备，万万没想到。最后还是拖到了考试前的半个小时。我真的是非常痛恨这种慌乱的感觉，但是一直也没能克服拖延这个坏毛病。上学以来每次考试都是临时抱佛脚，这次虽然复习的周期长了，但是效率仍然低得惊人，从12.14号开始复习以来，我至少浪费了百分之六十原本我计划用来复习的时间。如果不是今年痛改前非决定提前开始复习，我大概知道考完试也复习不完。

忙

发表于 2015-01-13   |   分类于 生活

明天选课，后天开始考试，虽然这次复习的还好，但是仍然有那种每每想起就心绞痛的感觉。非常非常的不爽，每次到了考试之前都会有这种荒废感奈何平时就是提不起尽头。这个学期比之前稍有了进展，至少这次我提前一个月开始了复习。不是自欺欺人地在自习室玩手机，而是实打实地复习。所以比以前多了一份坦然，我知道我真的有比别人多付出地东西，我也知道我会有收获。但是同时我也在警告自己，别做某些人一样的利己主义者，别认为帮助他人是浪费时间，别觉得别人不好是一件令人愉快的事情。谨此。

保持心情愉快。

一月十三日上午更新，选课成功抢到所有想要的课。很顺利，接下来就是考试了。再接再厉。

写在建立之初的话

发表于 2015-01-03   |   分类于 生活

这是我的第一篇日志，纪录一下我建立这个Blog，作为一个一向说得多做得少的人，能坚持作出这个东西我觉得也算是是一个不大不小的成就了。一开始做个网站只是觉得很帅，后来发现其实真的非常有意思，而且可以学到一些真正能够投入应用的东西。网站还有很多功能没有加，有时间一定会做。我暂时还不会怎么添加一个评论功能，感兴趣可以来我的主页逛逛imdiyi.com想聊聊可以给我发邮件。邮箱就在blog首页里面写好了。（在我还没宣传这个blog的阶段如果你们看到这篇文章那说明我们真的超级有缘，请联系我！）

最近临近大三期末，复习百无聊赖所以抽时间建立了这么一个blog，希望建立之后能够坚持使用下去，记录自己生活学习上面的点点滴滴。

对了，特别感谢西奇为我提供硬件支持，省了我一大笔钱，欢迎去踩这他的主页www.imxiqi.com

最后，时刻牢记，保持心情愉快。
