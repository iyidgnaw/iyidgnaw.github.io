贝叶斯推断及其互联网应用

作者：阮一峰



一、什么是贝叶斯推断

贝叶斯推断（Bayesian inference）是一种统计学方法，用来估计统计量的某种性质。

它是贝叶斯定理（Bayes’ theorem）的应用。英国数学家托马斯·贝叶斯（Thomas Bayes）在1763年发表的一篇论文中，首先提出了这个定理。



贝叶斯推断与其他统计学推断方法截然不同。它建立在主观判断的基础上，也就是说，你可以不需要客观证据，先估计一个值，然后根据实际结果不断修正。正是因为它的主观性太强，曾经遭到许多统计学家的诟病。

贝叶斯推断需要大量的计算，因此历史上很长一段时间，无法得到广泛应用。只有计算机诞生以后，它才获得真正的重视。人们发现，许多统计量是无法事先进行客观判断的，而互联网时代出现的大型数据集，再加上高速运算能力，为验证这些统计量提供了方便，也为应用贝叶斯推断创造了条件，它的威力正在日益显现。

迟来的2014总结

发表于 2015-01-24   |   分类于 生活

2014发生了太多事情，做了太多决定，有对的有错的但是我不后悔。

上半年让我至今记忆犹新的就两件事，第一个是打大班赛没出现，让我难受了好几天，我觉得真是我的锅，最后刚愎自用不传球，一个人拖垮了一个队，事实证明我一打全场就是怂，或者准确说是有劲使不出来的感觉，挺后悔的，没好好赢一场比赛。第二件事是恋爱谈崩了，看了一场同桌的你，直接导致长期经营不善的感情走向了终结，还是我的锅，没啥不能承认的。

我觉得分手这件事情在现在来看对我的影响的挺大的，毕竟也好几年了断断续续明的暗的一直身边有个她结果忽然有这么一天就这样分手了，怅然若失了一段时间。

下半年距离现在比较近所以记得很多事情，假期去了一趟淄博去了一趟青岛，我不得不说就这趟出游让我一直隐隐不快的心情彻底得到了释放，坐在海边的感觉真的很棒，就让海浪一下一下的冲刷，看着茫茫大海，我突然就觉得没什么事情是过不去的，而且我还有兄弟陪着我，多谢小贱的招待还有曾哥聪哥的陪伴。说到这不得不提曾哥，我觉得高中三年浑浑噩噩最让我一点遗憾都没有的事情就是交了曾哥这么一个朋友，可以让我把乐观之后留下的所有悲观一股脑倾诉出来，让我烦到无以复加的时候两瓶酒就释然，谢了。

于是就说到了开学前的一件可能会影响我一辈子的事情，是我活了二十年第一次做的人生选择。我决定出国留学。事情的起因就是不明不白的一次和西奇的碰面，先是兜了一个大圈才找到饭馆，没想到坐下三言两语我就不知怎么铁了心要出国了。不过我觉得这个决定并不草率，我觉得无论选择哪条路，工作也好，考研也好，出国也好，走下去，只要能让你成为一个更好的自己，就无所谓对错无所谓好坏，与其犹犹豫豫不如勇往直前。坦白说纠结的时间过长让我付出了太多代价，不知道以后的打算让我的大二浑浑噩噩不知道该干什么生活没有目标。大三上学期这个学期我的效率还是很低很低，但是至少我知道我要什么，经历了一次无比失败的GRE考试之后我意识到自己可能又给自己的拖延交了一次不菲的学费，我觉得这样下去我会毁了自己。所以我在下半学期毅然决然决定做个学霸。背单词我坚持不下来，看书我坚持不下来，每天健身我坚持不下来，减肥我坚持不下来，但是这次我觉得再不坚持下来我就没啥奔头了。

慌乱的今天

发表于 2015-01-16   |   分类于 生活

今天上午考通原，本以为妥妥的能在考前做好十足的准备，万万没想到。最后还是拖到了考试前的半个小时。我真的是非常痛恨这种慌乱的感觉，但是一直也没能克服拖延这个坏毛病。上学以来每次考试都是临时抱佛脚，这次虽然复习的周期长了，但是效率仍然低得惊人，从12.14号开始复习以来，我至少浪费了百分之六十原本我计划用来复习的时间。如果不是今年痛改前非决定提前开始复习，我大概知道考完试也复习不完。

我觉得拖延真的是我最大的敌人，因此我决心一定要打败他，最简单的办法，就是立刻去做该做的事情。其他一切都要为之让路，我觉得这十几天的效果不错，所有事情，包括吃饭睡觉都要给我制定的目标让路，没有任何理由可以阻碍定下的目标。虽然会熬夜，虽然会饿肚子，但是那些难受的感觉提醒我要抓紧做完该干的事情。同时，当我饿肚子当我不能睡觉的时候就意味着我又拖延了我自己本来的计划。

今天考完通原，下一科就是下周三的微机原理，看似复习时间很充足，但是下周三开始要考三科。所以我的时间其实远比这周还要紧迫，继续复习通原的效率，我觉得我一定能复习好。

周末结束之前我要结束微机原理的复习，不留任何问题，不要任何借口拖延到下周。现在真的真的不是松懈的时候。

付出一定有回报。

我坚信。
