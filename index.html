Martyr2’s Mega Project List

发表于 2016-05-03   |   分类于 学术

2016.5.3

今天开始重新回复python的精研，我仔细想了一下，还是不要操之过急地去学习c语言还有java了。虽然早晚是要学的，但是现在日常应用最多的就是python，还是很有必要精研一下python语言的。
在知乎里面找到了这个列表，里面有各个方面的python练手小程序，我决定每天写几个，数量不定，全凭自觉。这篇博文用来记录每个小程序的思考细节以及所用的相关的库以及各种小tips。
整个项目托管在了我的Github上。

Reverse a String:

代码的主体部分用了最基础的切片倒序，其实这个问题还是有挺大的发挥空间的，以后有时间可以再仔细自己写一个reverse函数。

def reverse(input):
    output = input[::-1]
Pig Latin：

我也不知道这个题目为什么叫这个名字，总感觉有点racial的意味在里面。。。然而这个题目并不是很难，总共就三种情况。所以我先建立了一个原音字母的列表 

origin = ['a','e','i','o','u']

然后再判断首字母是否为原音，如果是则直接返回。如果整个字符串中没有原音，那就尾端加ay返回，剩下一种就是最基本的情况了。

Count Vowels:

这个题目的本意应该也是自己写程序统计，但是对于这种已有轮子可以解决的问题，我确实没什么信心能写出更好的轮子，所以果断使用了Counter，

from collections import Counter

Counter库的应用其实挺广泛的，可以直接统计出list中的元素出现频率并且以一个特有的字典形式返回。明天研究一下Counter库的具体实现原理，看看能不能看懂（并没有信心看懂。。。）

Palindrome:

回文问题似乎是很经典的一个问题，之前刷leetcode的时候也遇到了好几道相关的问题。因为已经有切片反转的经验，所以分类之后直接判断前半段和后半段的字符串是否一致就可以了。
还是要吐槽一下sublime这个不能输入的问题，这样的话，我可能要重回pycharm的怀抱了。但是今天装了git tools和diff功能，感觉sublime的主题看着好舒服，好不想换。

2016三月杂记

发表于 2016-03-23   |   分类于 生活
科研

三月初收到了NEU的ad，喜出望外了一整天。然后就进入了无穷无尽的科研黑洞生活。上学期做出来的RNN，只是加了一点点功能，就调试不能通过了。好不容易调试通过了，结果又始终追不上原始的RNN。我觉得我在实习的将近一年时间里，发现了一个重要的事情，就是我不适合念博士。首先，我觉得没有一个明确目标是一件非常痛苦的事情，而前沿的课题，如果不是水课题的话，大多数是没有明确目标的，并且很多时候甚至是在做一些开创性的工作。这些工作没有前人的经验可以参考，也没有绝对的正确与错误，所以也就没有一个明确的终点可以作为目标，这种茫然的感觉是我非常不喜欢的。我不讨厌编程，我也不讨厌出bug，我现在甚至觉得出bug是一件非常开心的事情，因为我明确地知道，改掉一个bug就意味着我朝着正确的方向又迈进了一步。我所烦躁的是程序没有问题但是准确率就是没有提高，这种不知道问题出在哪里的感觉真是非常的差。

纠结的语言

最近两周的效率真是非常的低，因为每一个版本的代码都需要运行两天甚至三天才能看到一个相对稳定的结果，而在等待的过程中，完全是手足无措的。上周我利用等待的时间刷了一些Leetcode的算法题，但是刷完了之后又出现了新的坑。我现在可以比较熟练的使用python了，但是我发现python在处理一些任务时真的比java和c慢非常多。从leetcode的结果分析就可以看出。另外，有一道题我用python写完一直超时，同样的思路换到c直接就跑通了。于是乎上周我又开始学习c语言。抱着一本c primer看了好久，然而到了这周，因为魏总要周末复试所以就把书借给他了（话说这个书其实还是当年何尼玛借给我的），我也觉得这个书有点过于基础过于细致，这样看下去进度太慢。这样，我就面临了一如当初学习python时候的窘境－－没有一个明确的学习思路，我手头有很多资料，退一万步说我还可以看c的源代码。但是问题在于我没有一个任务导向，接下来的学习，是应该用c去刷算法题以此来熟悉c语言的数据结构？还是应该读一些经典的c的代码（比如python的底层，甚至是linux的底层）？这个问题是我这两天迫切需要解决的。

接下来的方向

上周我听了几节普林斯顿的算法课，让我纠结的点就在于，那个课程是java实现的。。。。因此这周犹豫不决的问题就是，到底接下来应该投入精力学c还是投入精力学java。首先要明确的一点是，数据结构和算法是肯定要继续搞下去的，但是具体用什么语言实现，这个就比较纠结了。我目前的想法是，以后去了neu肯定要选os课程，相对于java来说，我对c语言的理解其实非常薄弱，所以c的学习肯定是我目前绕不掉的一个环节。

生活习惯

上周开始重新启用了anydo，把日常需要做的事情和准备做的事情都放了进去，这个号的规划已经开始影响了我的生活。提醒我每天坚持记重。但其实任务的完成情况还是非常不理想，很多每天应该做的事情都没有消掉。同时，anydo的使用还让我放弃了之前一直坚持在写的纸质日程本，这是我接下来要改正的问题。日程本一定要坚持记下去，每周日一定要规划好下一周的安排，把大体上的安排做好。每天固定要做的就是刷算法题，刷算法课，学语言，调参数，学英语，听听力。这些事情一定要安排一个比较高的优先级，否则每天就浑浑噩噩地荒废过去了。另外，这两周节奏比较紧，同时身体也出了一点点小状况，导致健身的习惯也有所中断，这个是必须改正的。从今天开始，回去练腹肌，明天继续按照健身的计划恢复健身。

总结

写了好多好多，也算条分缕析地整理了一下思路。接下来要做的就是：

  1. 继续健身，每天记重。
  2. 做好规划，纸质和app同步进行，养成管理时间管理计划的的好习惯。
  3. 每天保证睡眠时间，weekday睡前不刷各类app。
  4. 每天按计划安排各项学习任务，保质保量完成，不要自欺欺人，不要拖延时间。
  5. 与人为善，调整好心态。
  6. 实时关注录取结果，安排好接下来的留学事宜。
  7. 科研工作上要有耐心，结果出不来也无可奈何，慢慢来。

进入2016之后的心态其实一直非常紧张，可能是2015的忙碌带来的后遗症，其实现在的事情都不是那么紧急了，所以希望接下来的一段时间，我能够放平心态，劳逸结合。过好每一天。

保持心情愉快

2015的总结

发表于 2016-02-21   |   分类于 生活

这一段时间在家百无聊赖，学习也学不进去，又没什么出去玩的由头，也提不起锻炼的兴致。每天在家不是胡思乱想就是没事找事.刚刚吃完午饭,我突然想起来，每年似乎都要写一个年终的总结。而2015的这一篇已经拖了太久，2016都已经快过去四分之一了，于是乎洗了个澡清清爽爽地坐到写字台前打开编辑器开始总结一下过去好久的2015. 坦白说这一年过的格外快，好像2014跨年的经历还历历在目一下子就到了今年的末尾。似乎上了大学之后每一年都没有什么头绪，想写点什么也觉得无从说起，所以就还是记一笔流水账想到什么就写什么吧，还好这个博客也只是为了记录一下我想写的东西，其实并没有人看。

2015其实发生了各种各样的事情，经历了托福GRE，经历了学校一个学期40分的狂轰乱炸的考试，经历了亲人去世。我觉得我似乎在21岁里强迫自己用一个本不属于自己的生活态度生活了一年。强迫自己变得所谓的成熟，不去胡思乱想一些有的没的，一心专注在所谓的前途上面。细细回想，我觉得有所得也有所失。大三之前我脑海里的大学生活从来就不是三点一线学习、吃饭、睡觉这样度过的，但是到了大三，感受到来自GPA的强大压力，以及出国的强烈愿望，我开始废寝忘食的学习。在很长的一段时间内获得一种充实感以及一种遗世而独立的优越感。充实感来源于我确实学会了很多东西，直到今天我也觉得能够把英语学好将是受用无穷的一件事情。然而刻苦的学习给我带来了一种不恰当的优越感，这种优越感来自于我和身边同学的比较，他们有的每天打游戏，有的每天陪女友，有的每天打球，导致我觉得能够静下心来坚持学习就是一件极其光荣且无比正确的事情。我甚至错过了一年一度的大班篮球赛，从2015年1.1开始整整半年一次篮球也没打，当时我甚至还隐隐感到得意。可是这种优越感在选定留学学校的时候彻底被打灭了。当我看到我的选校名单上的学校都是以前从没有听说过的学校，当我看到大家谈论的好学校我连申请的必要都没有的时候，我陷入了很长一段时间的沉沦。每天什么事情也不想做，虽然我没有所谓的名校情结，但是我总觉得自己值得更好的，因为我付出了努力。

这种失落感一直持续到了9月，后来我慢慢想清楚了两个事情，第一：一开始出国的目的就是为了成长为一个更好的自己，获得更多的锻炼机会。从这个角度说，哪怕我去了一个非常不好的学校，我也至少能够学会独立的生活，独立地处理日常生活中的方方面面。第二：我并没有比别人优秀在哪里。经过了仅仅一年的努力学习，在刻苦劲头仍未消退的阶段，我理所当然地认为我比别人掌握了更多的知识，有着更强的意志力。这种虚无缥缈的优越感随着身边人的变化逐渐被淡化了，考研狗们一个个地从吊儿郎当变成了拼命三郎，这使我意识到，我并没有什么出众的地方，每个人在为前途奋斗时都会表现出斗志，都会一反常态地忘我学习。我并不是唯一一个醒来的人，所谓的温水煮青蛙其实也只是身边的同学认为没有到刻苦努力的时候罢了。同时我也仔细反省了我这一年的刻苦学习，完全是为了偿还大一大二荒废的时间。我并没有比别人优秀在哪里，我只不过是在大一大二加倍地荒废了时间，而在大三再加倍地往回拉而已，说到底是一种还债行为。

当我认清了这两点，我发现我再一次看清了眼前的道路，就是继续努力补充自己，既然我不是一个比大众都优秀的人，既然我没有什么所谓的心智上的优势，那就继续刻苦地努力下去，总有一天可以达到我想要的状态。庆幸的是，我实习的模式识别实验室的老师和学长人都非常好，他们能够在学术上给我指导，也愿意为我的未来出谋划策，并且给了我足够的时间让我能够有条不紊的一步一步进行工作。其实下半年的科研工作可以说是相当的不顺，不过反思一下也是因为自己对知识的理解还不到位就强行上马了庞大的任务。老师和学长是没有责任和义务为你指出算法中的每一个问题的，说到底还是要靠自己。如果我们能在一开始就搞懂RNN的原理，一开始就加入BPTT的部分，也不会从9月一直改代码到12月。这一个惨痛的教训也从学术的角度告诉我，一切都要稳扎稳打，不能盲目追求速度。所以我们决定一步一个脚印地从最原始的矩阵分解开始，在经典数据集上实现一些基础算法，从FM，到BPR，再到RNN。一路走来我觉得清楚了很多，对整个公式的推导过程，对整个算法的实现和调试过程都有了更加清晰的认识。

可是现实似乎总是不能让我稳稳当当地度过每一天，度过了之前的盲目自信，度过了之前的迷茫，在我以为自己找到方向的时候，我却在自己最根本的方向选择上出现了含糊。

到底要不要出国？

我不可能一辈子当个学生，我总是要有走进社会独立生活的那一天的，最后也还是要回到祖国。那么像我这种就业导向型的出国留学，如果现阶段在国内能够找到一个满意的工作，还要不要出国留学呢？什么样的工作算是满意的工作呢？这一切的一切，直到今天我仍然没有想清楚，每天在家做着白日梦，想着我要是能拿到CMU的ad那么一切纠结也就迎刃而解了，想着我要是能拿到阿里的offer那么一切纠结也就迎刃而解了。然而幻想中这些事情，并没有到来，其实这才是理所当然的事情，现实又一次直截了当地让我认清了自己的定位。这几天似乎我想明白了一些事情，无论接下来是收到ad，还是在接下了春招中拿到offer，我都还是要努力成为一个更好的自己。每天在家刷剧看电影是实现不了这一目标的，该干的事情早晚都要去做。看着我空白了将近一个月的日程本，我感到有一种巨大的压迫感又回来了。虽然我不喜欢这种压迫感，但是我知道，对于现阶段的我来说，在压迫感下不断充实自己，远比带着盲目的自信和幻想无所事事要强。

说回2015，这一年的悲伤与忙碌都太过疯狂，付出与决定又似乎太过草率。唯一让我庆幸的就是收获了一段爱情。心动来的不早不晚，不疾不徐。到今天其实还没有100天，但是我觉得很踏实，好好过每一天，也许一不小心就会走过很远。反思这一百天的相处，我觉得我自己有点不够踏实。在这个热恋的阶段，陷入了一种过于依赖的状态，几乎要失去自己原有的生活轨迹。但我清楚的知道，在一段成熟健康的恋情里，我们每个人都应该先做好自己。从今天开始，我会自信地走好自己的路。不过坦白说，我还不知道我未来的路在哪里，也没有找到我的梦想是什么，但是我由衷的希望，当我走过这些年后回首时，当我有一天达到理想的目标时，我的身边还能有你。

这一年唯一值得骄傲的就是我认真地减了30斤，从195斤瘦到了现在162.从跑一圈就累，坚持到了甚至每天不跑10公里就浑身不自在，从深蹲50kg到深蹲110kg。

2016年我要精修Python，学好Java，认真看数据结构和算法，读完去年买了没时间读的书，继续学好英语，争取处理好与身边每一个朋友的关系，呵护好爱情。

2016是非常复杂的一年，这一年将要面临太多的抉择，有可能还会面临很多的分别，其实这一年也不过是漫漫人生路上普普通通的一年。这一年我给自己的寄语是：处变不惊。

2015过去了，带着太多理不清的思绪过去了。但愿，在已经开始了一段时间的2016年，所有爱我的人，和我爱的人，一切安好。

保持心情愉快
